============================================================
  COMMANDES TERMUX - VARIO ESP32
  Copier-coller dans Termux
============================================================


------------------------------------------------------------
  TESTER LES CAPTEURS DU TELEPHONE
------------------------------------------------------------

Lister tous les capteurs disponibles :
  termux-sensor -a -n 1

Tester le barometre directement :
  termux-sensor -s "TYPE_PRESSURE" -n 3

Aide termux-sensor :
  termux-sensor --help


------------------------------------------------------------
  VERIFIER LES FICHIERS
------------------------------------------------------------

Verifier que XCSoarData existe :
  ls ~/storage/shared/XCSoarData/

Verifier que le dossier lua existe :
  ls ~/storage/shared/XCSoarData/lua/

Verifier que vario_live.txt est cree (par Lua) :
  cat ~/storage/shared/XCSoarData/vario_live.txt

Surveiller vario_live.txt en temps reel :
  watch -n 1 cat ~/storage/shared/XCSoarData/vario_live.txt


------------------------------------------------------------
  COPIER LES FICHIERS DEPUIS TELECHARGEMENTS
------------------------------------------------------------

Copier gps_forward.py :
  cp ~/storage/downloads/gps_forward.py ~/

Creer le dossier lua et copier le script XCSoar :
  mkdir -p ~/storage/shared/XCSoarData/lua
  cp ~/storage/downloads/vario_export.lua ~/storage/shared/XCSoarData/lua/init.lua

Verifier que tout est en place :
  ls ~/gps_forward.py
  ls ~/storage/shared/XCSoarData/lua/init.lua


------------------------------------------------------------
  LANCER LE SCRIPT
------------------------------------------------------------

Lancer le forwarder GPS → ESP32 :
  python ~/gps_forward.py

Arreter :
  Ctrl + C

Relancer :
  python ~/gps_forward.py


------------------------------------------------------------
  VERIFIER LA CONNEXION ESP32
------------------------------------------------------------

Verifier que le WiFi est connecte sur FV_Displayboard :
  ping 192.168.2.1

Tester la connexion TCP a l'ESP32 :
  nc -zv 192.168.2.1 4353


------------------------------------------------------------
  VERIFIER GPS
------------------------------------------------------------

Tester la localisation GPS :
  termux-location -r once

Tester avec GPS force :
  termux-location -p gps -r once


------------------------------------------------------------
  INSTALLATION (si pas encore fait)
------------------------------------------------------------

Installer les dependances :
  pkg update -y && pkg install -y python termux-api

Autoriser le stockage :
  termux-setup-storage
